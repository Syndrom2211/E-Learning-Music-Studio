Sonar

Cakewalk SONAR adalah sebuah program komputer yang dibuat oleh Cakewalk untuk merekam, mengedit, mixing, mastering dan outputting audio. Versi terbaru dari perangkat lunak SONAR Home Studio 7, SONAR Home Studio 7 XL, SONAR 8,5 Studio Edition, SONAR Produser 8,5 Edition, dan SONAR LE. SONAR LE adalah versi OEM dibundel dengan hardware , semua versi lain dapat dibeli secara terpisah. Versi terakhir dari Software Cakewalk Sonar adalah SONAR X1 > SONAR x2 ( Family Bundle ).

SONAR adalah contoh dari Digital Audio Workstation (DAW). Seperti kebanyakan program atau software Digital Audio Workstation, bagi pengguna, fungsi dan kegunaan SONAR adalah untuk :

  • Merekam dan memanipulasi jumlah yang tidak terbatas multitrack audio digital (hanya dibatasi oleh hardware).
  • Secara komprehensif merekam dan memanipulasi data MIDI.
  • Menerapkan (aplikasi) setiap DirectX efek khusus, seperti reverb dan delay, banyak versi yang dibundel dengan efek.
  • Mengotomatiskan proses mixing audio.
  • Memanfaatkan virtual instrumen, seperti software synthesizers, software samplers, software drum machines.
  • Menghubungkan ke aplikasi multimedia lainnya dengan akurasi sampel melalui Rewire.
  • SONAR Producer Edition mencakup standar industri lengkap 64-bit MASTERING suite.
